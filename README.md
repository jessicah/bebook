# The Be Book / Haiku Book Amalgamation

This contains the work of creating the Be Book documentation in reStructuredText
& Markdown formats with Sphinx and MyST-Parser, and merging the Haiku Book
documentation into the Be Book.

It is hoped that this will result in a much more complete API documentation.

# `sphinx-output` Branch

This branch contains the reStructuredText and Markdown files for processing by
Sphinx to generate the final HTML for publishing to https://api.haiku.nz/.

These files are generated from the original BeBook source HTML files, turning
them into more semantically meaningful rST and MD formats, to make it easier to
modify and update the documentation after conversion has been completed.
